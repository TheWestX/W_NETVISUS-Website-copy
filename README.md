# W_NETVISUS Website (Practice 7)

This repository contains a single-page technical product website foundation for W_NETVISUS.

## How to publish on GitHub Pages
1. Push `index.html` to the repository root.
2. Go to **Settings → Pages**.
3. **Source**: Deploy from a branch
4. **Branch**: main / (root)
5. Save — GitHub will show your Pages URL.

No dependencies. Tailwind is loaded via CDN.
