W_NETVISUS • Network Visibility & Port Scanning

Go deep into your
network visibility.

W_NETVISUS is a modern network scanner with a 3D-like network map and a visual SONAR. It supports raw TCP SYN / UDP scanning via a C-extension and uses fingerprints for OS/service detection.

Quick Start

Minimal run instructions based on a typical repo layout (PyQt5 GUI + raw scan core + fingerprints).

Run locally

Linux/macOS example
1) Install dependencies
pip install PyQt5 netifaces
2) Run from project root
. venv/bin/activate
sudo PYTHONPATH=. python3 python_wrapper/scanner_gui_modern.py
Why sudo? Raw packet modes can require elevated permissions (depends on OS and scan mode).
